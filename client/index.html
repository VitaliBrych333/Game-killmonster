Убей всех монстров

Победи монстров!


Для управления можно использовать клавиши:
Tab + Enter - для выбора полей для ответа, применение магии;
Space - для выбора экрана рекордов/ проиграть звук в задаче с аудио;
1-8 - управление полями с Drag-n-drop;
0 - закрыть диалоговое окно;
Q-H - для выбора задач(магии)
